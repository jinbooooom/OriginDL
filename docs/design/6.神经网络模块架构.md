# 第六部分：神经网络模块架构

## 第17章：Module 系统设计

### 17.1 Module 基类架构

Module 是神经网络模块的基类，提供了参数管理、子模块管理、状态管理等核心功能。

#### 17.1.1 参数管理机制（parameters_ 映射）

**参数管理架构：**

```
┌─────────────────────────────────────┐
│  Module 参数管理                     │
├─────────────────────────────────────┤
│                                      │
│  parameters_ (unordered_map)        │
│    ├─> key: 参数名称 (string)        │
│    └─> value: 参数指针 (Parameter*)  │
│                                      │
│  设计特点：                          │
│    ✅ 使用指针避免拷贝                │
│    ✅ 便于统一管理                    │
│    ✅ 支持参数查找和更新              │
│                                      │
└─────────────────────────────────────┘
```

**参数注册流程：**

```
参数注册流程：
  ┌─────────────────────────────────────┐
  │  register_parameter(name, param)   │
  │    1. 检查是否已存在                │
  │    2. 存储参数指针                  │
  │    3. 添加到 parameters_ 映射       │
  └─────────────────────────────────────┘
```

#### 17.1.2 子模块管理（modules_ 映射）

**子模块管理架构：**

```

│                                      │
│  modules_ (unordered_map)           │
│    ├─> key: 模块名称 (string)        │
│    └─> value: unique_ptr<Module>     │
│                                      │
│  设计特点：                          │
│    ✅ 使用 unique_ptr 管理所有权      │
│    ✅ 支持嵌套模块结构                │
│    ✅ 自动管理子模块生命周期           │
│                                      │
└─────────────────────────────────────┘
```

**嵌套模块结构：**

```
嵌套模块示例：
```

### 17.2 参数注册机制

#### 17.2.1 register_parameter() 设计

**参数注册架构：**

```
参数注册流程：
  ┌─────────────────────────────────────┐
  │  register_parameter(name, param)   │
  │    1. 检查是否已存在                │
  │    2. 存储参数指针到映射             │
  └─────────────────────────────────────┘
              │
              ▼
  ┌─────────────────────────────────────┐
  │  parameters_ 映射                   │
  │    "weight" ──> &weight_           │
  │    "bias" ──> &bias_               │
  └─────────────────────────────────────┘
```

**参数注册示例：**

```
Linear 层参数注册：
  ┌─────────────────────────────────────┐
  │  Linear                            │
  │    ├─> weight_ (Parameter)         │
  │    │     └─> register("weight")    │
  │    └─> bias_ (Parameter)           │
  │          └─> register("bias")      │
  └─────────────────────────────────────┘
```

**关键点：**
- 参数必须是 Module 的成员变量
- 注册时传递引用，存储指针
- 名称必须唯一

#### 17.2.2 参数命名与路径

**命名规则：**
- 同一模块内：参数名称唯一
- 不同模块：可以使用相同名称
- 完整路径：模块路径 + 参数名称

**路径构建：**
```cpp
std::unordered_map<std::string, Parameter *> Module::named_parameters(const std::string &prefix) {
    std::unordered_map<std::string, Parameter *> named_params;
    
    // 当前模块的参数
    for (auto &[name, param] : parameters_) {
        std::string full_name = prefix.empty() ? name : prefix + "." + name;
        named_params[full_name] = param;
    }
    
    // 子模块的参数（递归）
    for (auto &[name, module] : modules_) {
        std::string module_prefix = prefix.empty() ? name : prefix + "." + name;
        auto sub_params = module->named_parameters(module_prefix);
        named_params.insert(sub_params.begin(), sub_params.end());
    }
    
    return named_params;
}
```

**示例：**
```
模型结构：
MyModel
  ├─ linear1 (Linear)
  │    ├─ weight
  │    └─ bias
  └─ linear2 (Linear)
       ├─ weight
       └─ bias

参数路径：
- "linear1.weight"
- "linear1.bias"
- "linear2.weight"
- "linear2.bias"
```

**获取所有参数：**


**获取命名参数：**
```cpp

```

**访问特定参数：**

### 17.3 StateDict 设计

#### 17.3.1 PyTorch 风格的状态字典

**定义：**
```cpp
using StateDict = std::unordered_map<std::string, Tensor>;
```

**特点：**
- 键：参数完整路径（字符串）
- 值：参数张量（Tensor）
- 与 PyTorch 的 `state_dict()` 行为一致

**获取状态字典：**
```cpp
StateDict Module::state_dict() const {
    StateDict state_dict;
    auto named_params = named_parameters();
    
    for (const auto &[name, param] : named_params) {
        state_dict[name] = *param;  // Parameter 继承自 Tensor
    }
    
    return state_dict;
}
```

**使用：**
```cpp
auto state_dict = model.state_dict();
// state_dict["linear1.weight"] -> Tensor
```

#### 17.3.2 参数序列化

**保存：**
```cpp
void save(const std::string &filepath, const Module::StateDict &state_dict);
```

**加载：**
```cpp
Module::StateDict load(const std::string &filepath);
```

**文件格式：**
- 使用自定义格式（.odl）
- 存储参数名称和数值
- 支持压缩（可选）

#### 17.3.3 模型保存与加载

**保存模型：**
```cpp
// 方法1：保存状态字典
auto state_dict = model.state_dict();
save("model.odl", state_dict);

// 方法2：直接保存（如果 Module 提供）
model.save("model.odl");
```

**加载模型：**
```cpp
// 方法1：加载状态字典
auto state_dict = load("model.odl");
model.load_state_dict(state_dict);

// 方法2：直接加载（便捷方法）
model.load("model.odl");
```

**load_state_dict() 实现：**
```cpp
void Module::load_state_dict(const StateDict &state_dict, bool strict = true) {
    auto named_params = named_parameters();
    std::set<std::string> loaded_keys;
    
    // 加载参数
    for (auto &[name, param] : named_params) {
        auto it = state_dict.find(name);
        if (it != state_dict.end()) {
            // 检查形状
            if (param->shape() != it->second.shape()) {
                THROW_RUNTIME_ERROR("Shape mismatch for parameter '{}'", name);
            }
            // 更新参数
            *param = Parameter(it->second);
            loaded_keys.insert(name);
        } else if (strict) {
            THROW_RUNTIME_ERROR("Missing parameter '{}'", name);
        }
    }
    
    // 检查未使用的键（strict 模式）
    if (strict) {
        for (const auto &[key, value] : state_dict) {
            if (loaded_keys.find(key) == loaded_keys.end()) {
                THROW_RUNTIME_ERROR("Unexpected parameter '{}'", key);
            }
        }
    }
}
```

**strict 模式：**
- **strict = true**：严格匹配，所有参数必须存在且匹配
- **strict = false**：允许部分参数缺失，用于迁移学习等场景

### 17.4 Sequential 容器

#### 17.4.1 层的有序管理

**设计：**
```cpp
class Sequential : public Module {
private:
    std::vector<std::unique_ptr<Module>> modules_;
    
public:
    void add(std::unique_ptr<Module> module);
    Tensor forward(const Tensor &input) override;
};
```

**特点：**
- 使用 `vector` 存储模块（有序）
- 使用 `unique_ptr` 管理所有权
- 支持索引访问

**添加模块：**
```cpp
void Sequential::add(std::unique_ptr<Module> module) {
    modules_.push_back(std::move(module));
}
```

**使用示例：**
```cpp
Sequential model;
model.add(std::make_unique<Linear>(784, 128));
model.add(std::make_unique<ReLU>());
model.add(std::make_unique<Linear>(128, 10));
```

#### 17.4.2 前向传播流程

**实现：**
```cpp
Tensor Sequential::forward(const Tensor &input) {
    Tensor output = input;
    
    // 依次通过所有模块
    for (auto &module : modules_) {
        output = module->forward(output);
    }
    
    return output;
}
```

**流程：**
```
input
  ↓
module[0].forward()
  ↓
module[1].forward()
  ↓
module[2].forward()
  ↓
output
```

**特点：**
- 顺序执行
- 每个模块的输出作为下一个模块的输入
- 简单直观

#### 17.4.3 参数自动注册

**实现：**
```cpp
std::vector<Parameter *> Sequential::parameters() {
    std::vector<Parameter *> params;
    
    // 收集基类参数（如果有）
    auto base_params = Module::parameters();
    params.insert(params.end(), base_params.begin(), base_params.end());
    
    // 收集所有子模块的参数
    for (auto &module : modules_) {
        auto sub_params = module->parameters();
        params.insert(params.end(), sub_params.begin(), sub_params.end());
    }
    
    return params;
}
```

**自动注册：**
- Sequential 的子模块自动注册到 `modules_`
- 子模块的参数自动收集
- 用户无需手动注册

## 第18章：Layer 实现架构

### 18.1 Layer 基类设计

#### 18.1.1 Layer 接口定义

**设计：**
```cpp
class Layer : public Module {
public:
    virtual Tensor forward(const Tensor &input) = 0;
};
```

**特点：**
- 继承自 Module
- 提供前向传播接口
- 支持参数管理

**与 Module 的关系：**
- Layer 是 Module 的特化
- Layer 专门用于神经网络层
- Module 更通用，可以表示任意模块

#### 18.1.2 Parameter 管理

**Parameter 类：**
```cpp
class Parameter : public Tensor {
public:
    using Tensor::Tensor;  // 继承所有构造函数
    Parameter(const Tensor &tensor) : Tensor(tensor) {}
};
```

**特点：**
- Parameter 继承自 Tensor
- 可以像 Tensor 一样使用
- 用于标识需要优化的参数

**使用：**
```cpp
class Linear : public Layer {
private:
    Parameter weight_;  // 权重参数
    Parameter bias_;    // 偏置参数
};
```

#### 18.1.3 前向传播接口

**接口：**
```cpp
virtual Tensor forward(const Tensor &input) = 0;
```

**实现模式：**
```cpp
Tensor Linear::forward(const Tensor &input) override {
    // 1. 验证输入
    // 2. 执行计算
    auto output = mat_mul(input, weight_);
    if (use_bias_) {
        output = output + bias_;
    }
    // 3. 返回结果
    return output;
}
```

**特点：**
- 纯虚函数，强制子类实现
- 输入输出都是 Tensor
- 支持自动求导

### 18.2 Linear 层实现

#### 18.2.1 权重与偏置参数

**参数定义：**
```cpp
class Linear : public Layer {
private:
    Parameter weight_;  // 权重：(in_features, out_features)
    Parameter bias_;    // 偏置：(1, out_features) 或 (out_features,)
    int in_features_;
    int out_features_;
    bool use_bias_;
};
```

**参数形状：**
- **weight_**: `(in_features, out_features)`
- **bias_**: `(1, out_features)` 或 `(out_features,)`
- 支持不使用偏置（use_bias_ = false）

#### 18.2.2 前向传播（matmul + add）

**实现：**
```cpp
Tensor Linear::forward(const Tensor &input) {
    // 1. 矩阵乘法：y = input * weight
    auto output = mat_mul(input, static_cast<const Tensor &>(weight_));
    
    // 2. 添加偏置（如果需要）
    if (use_bias_) {
        output = output + static_cast<const Tensor &>(bias_);
    }
    
    return output;
}
```

**数学公式：**
```
y = x * W^T + b
```

**形状变换：**
- 输入：`(batch_size, in_features)`
- 权重：`(in_features, out_features)`
- 输出：`(batch_size, out_features)`

#### 18.2.3 反向传播（梯度计算）

**自动求导：**
- 前向传播时自动构建计算图
- backward() 时自动计算梯度
- 不需要手动实现反向传播

**梯度计算：**
- **gx** = gy * weight（输入梯度）
- **gweight** = gy^T * x（权重梯度）
- **gbias** = sum(gy, dim=0)（偏置梯度）

#### 18.2.4 参数初始化

**权重初始化（Xavier）：**
```cpp
Parameter Linear::init_weight() {
    // Xavier 初始化：scale * randn
    float scale = std::sqrt(1.0f / static_cast<float>(in_features_));
    auto weight_tensor = Tensor::randn({in_features_, out_features_});
    auto scaled_weight = weight_tensor * Scalar(scale);
    return Parameter(scaled_weight);
}
```

**偏置初始化：**
```cpp
Parameter Linear::init_bias() {
    // 初始化为零
    auto bias_tensor = Tensor::zeros({1, out_features_});
    return Parameter(bias_tensor);
}
```

**重置参数：**
```cpp
void Linear::reset_parameters() {
    weight_ = init_weight();
    if (use_bias_) {
        bias_ = init_bias();
    }
}
```

### 18.3 Conv2d 层实现

#### 18.3.1 卷积操作原理

**卷积公式：**
```
output[b, c_out, h, w] = sum(
    input[b, c_in, h+i, w+j] * weight[c_out, c_in, i, j]
) + bias[c_out]
```

**参数：**
- **weight_**: `(out_channels, in_channels, kernel_h, kernel_w)`
- **bias_**: `(out_channels,)`
- **stride**: 步长
- **padding**: 填充

#### 18.3.2 im2col 实现

**im2col 方法：**
- 将卷积转换为矩阵乘法
- 提高计算效率
- 增加内存使用

**流程：**
```
输入 (N, C, H, W)
  ↓ im2col
列矩阵 (N*H_out*W_out, C*K*K)
  ↓ 矩阵乘法
输出 (N*H_out*W_out, out_channels)
  ↓ reshape
输出 (N, out_channels, H_out, W_out)
```

#### 18.3.3 前向传播实现

**实现：**
```cpp
Tensor Conv2d::forward(const Tensor &input) {
    // 1. im2col 转换
    auto im2col_result = im2col(input, kernel_size_, stride_, padding_);
    
    // 2. 矩阵乘法
    auto weight_flat = flatten_weight(weight_);
    auto matmul_result = matmul(im2col_result, weight_flat);
    
    // 3. reshape 回输出形状
    auto output = reshape(matmul_result, output_shape_);
    
    // 4. 添加偏置
    if (use_bias_) {
        output = output + bias_;
    }
    
    return output;
}
```

#### 18.3.4 反向传播实现

**梯度计算：**
- **ginput** = col2im(goutput * weight^T)
- **gweight** = im2col(input)^T * goutput
- **gbias** = sum(goutput, dims)

**实现：**
```cpp
std::vector<Tensor> Conv2d::backward(const std::vector<Tensor> &gys) {
    // 梯度计算较复杂，需要 im2col 的逆操作 col2im
    // ...
}
```

### 18.4 其他层实现

#### 18.4.1 BatchNorm：训练/推理模式切换

**前向传播：**
```cpp
Tensor BatchNorm::forward(const Tensor &input) {
    if (training_) {
        // 训练模式：使用当前 batch 的统计量
        auto mean = mean(input, dims_);
        auto var = variance(input, dims_);
        
        // 更新全局统计量（移动平均）
        running_mean_ = momentum_ * running_mean_ + (1 - momentum_) * mean;
        running_var_ = momentum_ * running_var_ + (1 - momentum_) * var;
        
        // 归一化
        auto x_norm = (input - mean) / sqrt(var + eps_);
        return x_norm * weight_ + bias_;
    } else {
        // 推理模式：使用全局统计量
        auto x_norm = (input - running_mean_) / sqrt(running_var_ + eps_);
        return x_norm * weight_ + bias_;
    }
}
```

**关键点：**
- 训练模式：计算当前 batch 的均值和方差
- 推理模式：使用保存的全局统计量
- 通过 `training_` 标志切换

#### 18.4.2 Pooling：最大池化/平均池化

**最大池化：**
```cpp
Tensor MaxPool2d::forward(const Tensor &input) {
    // 保存索引（用于反向传播）
    auto [output, indices] = max_pool2d(input, kernel_size_, stride_);
    saved_indices_ = indices;
    return output;
}
```

**平均池化：**
```cpp
Tensor AvgPool2d::forward(const Tensor &input) {
    return avg_pool2d(input, kernel_size_, stride_);
}
```

**反向传播：**
- **最大池化**：梯度只传播到最大值位置
- **平均池化**：梯度平均分配到所有位置

#### 18.4.3 Activation：ReLU、Sigmoid、Softmax

**作为 Layer：**
```cpp
class ReLU : public Layer {
public:
    Tensor forward(const Tensor &input) override {
        return relu(input);  // 调用算子
    }
};
```

**特点：**
- 激活函数层通常没有参数
- 直接调用对应的算子
- 支持自动求导

```cpp
struct ParameterGroup {
```

```cpp
// 创建优化器
SGD optimizer(model.parameters(), 0.01);  // 学习率 0.01

// 更新学习率
optimizer.set_lr(0.001);
```

#### 19.1.3 Hook 机制

- **after_step**：更新参数后调用

**实现：**

```cpp
class Optimizer {
protected:
```

### 19.2 SGD 实现

#### 19.2.1 梯度下降算法

**更新公式：**
```
param = param - lr * grad
```

**实现：**
```cpp
void SGD::step() {
    for (auto &group : param_groups_) {
        float lr = group.lr_;
        
        for (auto *param : group.parameters_) {
            if (param->grad().impl_) {
                // 更新参数
                *param = *param - lr * param->grad();
            }
        }
    }
}
```

#### 19.2.2 动量机制

**更新公式：**
```
v = momentum * v - lr * grad
param = param + v
```

**实现：**
```cpp
class SGD {
private:
    std::unordered_map<Parameter *, Tensor> velocity_;  // 速度
    
public:
    void step() {
        for (auto &group : param_groups_) {
            for (auto *param : group.parameters_) {
                // 初始化速度
                if (velocity_.find(param) == velocity_.end()) {
                    velocity_[param] = Tensor::zeros(param->shape());
                }
                
                // 更新速度
                velocity_[param] = momentum_ * velocity_[param] - lr * param->grad();
                
                // 更新参数
                *param = *param + velocity_[param];
            }
        }
    }
};
```

#### 19.2.3 权重衰减

**更新公式：**
```
grad = grad + weight_decay * param
param = param - lr * grad
```

**实现：**
```cpp
void SGD::step() {
    for (auto &group : param_groups_) {
        for (auto *param : group.parameters_) {
            auto grad = param->grad();
            
            // 添加权重衰减
            if (weight_decay_ > 0) {
                grad = grad + weight_decay_ * (*param);
            }
            
            // 更新参数
            *param = *param - lr * grad;
        }
    }
}
```

### 19.3 Adam 实现

#### 19.3.1 一阶矩估计

**更新公式：**
```
m = beta1 * m + (1 - beta1) * grad
```

**实现：**
```cpp
class Adam {
private:
    std::unordered_map<Parameter *, Tensor> m_;  // 一阶矩
    
public:
    void step() {
        for (auto *param : parameters_) {
            // 初始化
            if (m_.find(param) == m_.end()) {
                m_[param] = Tensor::zeros(param->shape());
            }
            
            // 更新一阶矩
            m_[param] = beta1_ * m_[param] + (1 - beta1_) * param->grad();
        }
    }
};
```

#### 19.3.2 二阶矩估计

**更新公式：**
```
v = beta2 * v + (1 - beta2) * grad^2
```

**实现：**
```cpp
class Adam {
private:
    std::unordered_map<Parameter *, Tensor> v_;  // 二阶矩
    
public:
    void step() {
        for (auto *param : parameters_) {
            // 初始化
            if (v_.find(param) == v_.end()) {
                v_[param] = Tensor::zeros(param->shape());
            }
            
            // 更新二阶矩
            auto grad_sq = square(param->grad());
            v_[param] = beta2_ * v_[param] + (1 - beta2_) * grad_sq;
        }
    }
};
```

#### 19.3.3 自适应学习率

**更新公式：**
```
m_hat = m / (1 - beta1^t)
v_hat = v / (1 - beta2^t)
param = param - lr * m_hat / (sqrt(v_hat) + eps)
```

**实现：**
```cpp
void Adam::step() {
    t_++;  // 时间步
    
    for (auto *param : parameters_) {
        // 偏差修正
        auto m_hat = m_[param] / (1 - pow(beta1_, t_));
        auto v_hat = v_[param] / (1 - pow(beta2_, t_));
        
        // 更新参数
        auto update = m_hat / (sqrt(v_hat) + eps_);
        *param = *param - lr_ * update;
    }
}
```

**特点：**
- 自适应学习率：根据梯度历史调整
- 偏差修正：修正初始偏差
- 适合非平稳目标函数

### 19.4 Hook 系统

#### 19.4.1 Hook 接口设计

**Hook 类型：**
```cpp
using Hook = std::function<void(Parameter *, Tensor &)>;
```

**注册 Hook：**
```cpp
void Optimizer::register_hook(const Hook &hook) {
    hooks_.push_back(hook);
}
```

**调用 Hook：**
```cpp
void Optimizer::step() {
    for (auto *param : parameters_) {
        auto grad = param->grad();
        
        // 调用 hook（可以修改梯度）
        for (auto &hook : hooks_) {
            hook(param, grad);
        }
        
        // 更新参数
        *param = *param - lr_ * grad;
    }
}
```

#### 19.4.2 梯度裁剪实现

**梯度裁剪：**
```cpp
void gradient_clipping(Parameter *param, Tensor &grad, float max_norm) {
    float norm = compute_norm(grad);
    if (norm > max_norm) {
        grad = grad * (max_norm / norm);  // 缩放梯度
    }
}
```

**使用：**
```cpp
auto clip_hook = [max_norm](Parameter *param, Tensor &grad) {
    gradient_clipping(param, grad, max_norm);
};
optimizer.register_hook(clip_hook);
```

#### 19.4.3 学习率调度器

**调度器接口：**
```cpp
class LRScheduler {
public:
    virtual float get_lr(int epoch) = 0;
};
```

**StepLR 实现：**
```cpp
class StepLR : public LRScheduler {
private:
    float initial_lr_;
    int step_size_;
    float gamma_;
    
public:
    float get_lr(int epoch) override {
        return initial_lr_ * pow(gamma_, epoch / step_size_);
    }
};
```

**使用：**
```cpp
auto scheduler = StepLR(0.01, 10, 0.1);  // 每10个epoch衰减0.1倍

for (int epoch = 0; epoch < num_epochs; ++epoch) {
    float lr = scheduler.get_lr(epoch);
    optimizer.set_lr(lr);
    // 训练...
}
```
