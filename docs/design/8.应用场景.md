## 第23章：训练场景

### 23.1 线性回归

#### 23.1.1 问题定义

**线性回归问题架构：**

```
┌─────────────────────────────────────┐
│  线性回归问题                        │
├─────────────────────────────────────┤
│                                      │
│  问题：学习线性关系                  │
│    y = w * x + b                    │
│                                      │
│  目标：找到最优的 w 和 b             │
│    使得预测值与真实值的误差最小       │
│                                      │
│  损失函数：                          │
│    MSE = mean((y_pred - y_true)^2)  │
│                                      │
│  优化目标：                          │
│    minimize MSE(w, b)               │
│                                      │
└─────────────────────────────────────┘
```

#### 23.1.2 模型实现

**线性回归模型架构：**

```
┌─────────────────────────────────────┐
│  线性回归模型                        │
├─────────────────────────────────────┤
│                                      │
│  预测函数：                          │
│    y = x * w + b                    │
│                                      │
│  损失函数：                          │
│    MSE = mean((y_pred - y_true)^2)  │
│                                      │
│  参数：                              │
│    w (权重)                          │
│    b (偏置)                          │
│                                      │
└─────────────────────────────────────┘
```

**计算图结构：**

```
线性回归计算图：
  x ──┐
       ├─> [matmul] ──> x*w ──┐
  w ──┘                        ├─> [add] ──> y_pred
                                │
  b ────────────────────────────┘
  
  y_true ──┐
           ├─> [sub] ──> diff ──> [pow] ──> [sum] ──> [mean] ──> loss
  y_pred ──┘
```

#### 23.1.3 训练循环

**训练循环架构：**

```
┌─────────────────────────────────────┐
│  训练循环流程                        │
├─────────────────────────────────────┤
│                                      │
│  1. 初始化参数                       │
│     w = 0, b = 0                    │
│                                      │
│  2. 训练循环（for each iteration）   │
│     ├─> 清零梯度                     │
│     ├─> 前向传播                     │
│     ├─> 计算损失                     │
│     ├─> 反向传播                     │
│     └─> 更新参数                     │
│                                      │
└─────────────────────────────────────┘
```

**训练流程详细图：**

```
训练流程：
  ┌─────────────────────────────────────┐
  │  1. 清零梯度                        │
  │     w.clear_grad()                  │
  │     b.clear_grad()                  │
  └─────────────────────────────────────┘
              │
              ▼
  ┌─────────────────────────────────────┐
  │  2. 前向传播                        │
  │     y_pred = x * w + b             │
  │     loss = MSE(y, y_pred)          │
  └─────────────────────────────────────┘
              │
              ▼
  ┌─────────────────────────────────────┐
  │  3. 反向传播                        │
  │     loss.backward()                 │
  │     └─> 计算梯度                    │
  └─────────────────────────────────────┘
              │
              ▼
  ┌─────────────────────────────────────┐
  │  4. 更新参数                        │
  │     w = w - lr * w.grad()          │
  │     b = b - lr * b.grad()          │
  └─────────────────────────────────────┘
```

#### 23.1.4 梯度下降

**梯度下降更新架构：**

```
┌─────────────────────────────────────┐
│  梯度下降更新规则                    │
├─────────────────────────────────────┤
│                                      │
│  w = w - lr * ∂loss/∂w              │
│  b = b - lr * ∂loss/∂b              │
│                                      │
│  其中：                              │
│    lr: 学习率                        │
│    ∂loss/∂w: w 的梯度               │
│    ∂loss/∂b: b 的梯度               │
│                                      │
└─────────────────────────────────────┘
```

**梯度下降流程：**

```
梯度下降流程：
  ┌─────────────────────────────────────┐
  │  1. 自动求导计算梯度                 │
  │     loss.backward()                 │
  │     └─> 计算 ∂loss/∂w, ∂loss/∂b    │
  └─────────────────────────────────────┘
              │
              ▼
  ┌─────────────────────────────────────┐
  │  2. 梯度下降更新参数                 │
  │     w = w - lr * w.grad()          │
  │     b = b - lr * b.grad()          │
  └─────────────────────────────────────┘
```

**关键点：**

```
┌─────────────────────────────────────┐
│  梯度下降关键点                      │
├─────────────────────────────────────┤
│                                      │
│  ✅ 使用自动求导计算梯度              │
│                                      │
│  ✅ 手动实现梯度下降                  │
│     （也可以使用优化器）              │
│                                      │
│  ✅ 每次迭代前清零梯度                │
│                                      │
└─────────────────────────────────────┘
```

#### 23.2.3 CNN 实现

**CNN 结构：**
```
输入 (batch_size, 1, 28, 28)
  ↓
Conv2d(1, 64, 3x3) + BatchNorm2d + ReLU + MaxPool2d
  ↓
Conv2d(64, 128, 3x3) + BatchNorm2d + ReLU + MaxPool2d
  ↓
Flatten
  ↓
Linear(7*7*128, 128) + ReLU
  ↓
Linear(128, 10)
  ↓
输出 (batch_size, 10)
```

**实现：**
```cpp
class SimpleCNN : public Module {
private:
    std::unique_ptr<nn::Conv2d> conv1_;
    std::unique_ptr<nn::BatchNorm2d> bn1_;
    std::unique_ptr<nn::ReLU> relu1_;
    std::unique_ptr<nn::MaxPool2d> maxpool1_;
    std::unique_ptr<nn::Conv2d> conv2_;
    std::unique_ptr<nn::BatchNorm2d> bn2_;
    std::unique_ptr<nn::ReLU> relu2_;
    std::unique_ptr<nn::MaxPool2d> maxpool2_;
    std::unique_ptr<nn::Flatten> flatten_;
    std::unique_ptr<nn::Linear> fc1_;
    std::unique_ptr<nn::ReLU> relu3_;
    std::unique_ptr<nn::Linear> fc2_;
    
public:
    SimpleCNN()
        : conv1_(std::make_unique<nn::Conv2d>(1, 64, {3, 3}, {1, 1}, {1, 1}, true)),
          bn1_(std::make_unique<nn::BatchNorm2d>(64)),
          relu1_(std::make_unique<nn::ReLU>()),
          maxpool1_(std::make_unique<nn::MaxPool2d>({2, 2}, {2, 2}, {0, 0})),
          conv2_(std::make_unique<nn::Conv2d>(64, 128, {3, 3}, {1, 1}, {1, 1}, true)),
          bn2_(std::make_unique<nn::BatchNorm2d>(128)),
          relu2_(std::make_unique<nn::ReLU>()),
          maxpool2_(std::make_unique<nn::MaxPool2d>({2, 2}, {2, 2}, {0, 0})),
          flatten_(std::make_unique<nn::Flatten>()),
          fc1_(std::make_unique<nn::Linear>(7 * 7 * 128, 128, true)),
          relu3_(std::make_unique<nn::ReLU>()),
          fc2_(std::make_unique<nn::Linear>(128, 10, true))
    {
        // 注册所有子模块
        register_module("conv1", std::move(conv1_));
        register_module("bn1", std::move(bn1_));
        // ...
    }
    
    Tensor forward(const Tensor &input) override {
        auto x = input;
        x = conv1_->forward(x);
        x = bn1_->forward(x);
        x = relu1_->forward(x);
        x = maxpool1_->forward(x);
        x = conv2_->forward(x);
        x = bn2_->forward(x);
        x = relu2_->forward(x);
        x = maxpool2_->forward(x);
        x = flatten_->forward(x);
        x = fc1_->forward(x);
        x = relu3_->forward(x);
        x = fc2_->forward(x);
        return x;
    }
};
```

#### 23.2.4 训练与评估

**训练循环：**
```cpp
// 创建模型和优化器
SimpleCNN model;
Adam optimizer(model.parameters(), 0.001);

// 训练模式
model.train();

// 训练循环
for (int epoch = 0; epoch < num_epochs; ++epoch) {
    for (auto &batch : train_loader) {
        auto images = batch.first;
        auto labels = batch.second;
        
        // 前向传播
        auto outputs = model(images);
        auto loss = softmax_cross_entropy(outputs, labels);
        
        // 反向传播
        optimizer.zero_grad();
        loss.backward();
        optimizer.step();
    }
    
    // 评估
    model.eval();
    float accuracy = evaluate(model, test_loader);
    std::cout << "Epoch " << epoch << ", Accuracy: " << accuracy << std::endl;
}
```

**评估函数：**
```cpp
float evaluate(Module &model, DataLoader &loader) {
    model.eval();
    int correct = 0;
    int total = 0;
    
    for (auto &batch : loader) {
        auto images = batch.first;
        auto labels = batch.second;
        
        // 前向传播
        auto outputs = model(images);
        
        // 计算准确率
        auto pred = argmax(outputs, 1);
        for (size_t i = 0; i < pred.shape()[0]; ++i) {
            if (pred[i] == labels[i]) {
                correct++;
            }
            total++;
        }
    }
    
    return static_cast<float>(correct) / total;
}
```

#### 23.3.1 残差块设计

**残差块结构：**
```

```cpp
class ResidualBlock : public Module {
private:
    std::unique_ptr<nn::Conv2d> conv1_;
    std::unique_ptr<nn::BatchNorm2d> bn1_;
    std::unique_ptr<nn::ReLU> relu1_;
    std::unique_ptr<nn::Conv2d> conv2_;
    std::unique_ptr<nn::BatchNorm2d> bn2_;
    std::unique_ptr<nn::ReLU> relu2_;
    
public:
    ResidualBlock(int in_channels, int out_channels) {
        // 创建层...
    }
    
    Tensor forward(const Tensor &input) override {
        auto x = input;
        
        // 第一个卷积块
        x = conv1_->forward(x);
        x = bn1_->forward(x);
        x = relu1_->forward(x);
        
        // 第二个卷积块
        x = conv2_->forward(x);
        x = bn2_->forward(x);
        
        // 残差连接
        x = x + input;  // 需要确保形状匹配
        
        // 激活
        x = relu2_->forward(x);
        
        return x;
    }
};
```

#### 24.1.2 图像预处理

**预处理流程：**
```
原始图像 (BGR, 任意尺寸)
  ↓
Resize 到 224x224
  ↓
BGR → RGB
  ↓
转换为 float32
  ↓
归一化到 [0, 1]
  ↓
标准化（减去均值，除以标准差）
  ↓
转换为 Tensor (1, 3, 224, 224)
```

**实现：**
```cpp
#ifdef OPENCV_FOUND
Tensor preprocess_image(const cv::Mat& image, const Device& device) {
    const int input_h = 224;
    const int input_w = 224;
    const int input_c = 3;
    
    // 1. Resize
    cv::Mat resize_image;
    cv::resize(image, resize_image, cv::Size(input_w, input_h));
    
    // 2. BGR to RGB
    cv::Mat rgb_image;
    cv::cvtColor(resize_image, rgb_image, cv::COLOR_BGR2RGB);
    
    // 3. 转换为 float32
    cv::Mat float_image;
    rgb_image.convertTo(float_image, CV_32FC3);
    
    // 4. 分离通道并转换为 Tensor 格式
    std::vector<cv::Mat> split_images;
    cv::split(float_image, split_images);
    
    std::vector<float> input_data(input_c * input_h * input_w);
    for (int c = 0; c < input_c; ++c) {
        const cv::Mat& channel = split_images[c];
        cv::Mat transposed = channel.t();
        for (int h = 0; h < input_h; ++h) {
            for (int w = 0; w < input_w; ++w) {
                input_data[c * input_h * input_w + h * input_w + w] = 
                    transposed.at<float>(w, h);
            }
        }
    }
    
    Shape input_shape{1, static_cast<size_t>(input_c), 
                     static_cast<size_t>(input_h), static_cast<size_t>(input_w)};
    Tensor result = Tensor(input_data, input_shape, 
                          dtype(DataType::kFloat32).device(device));
    
    // 5. 归一化到 [0, 1]
    auto data_vec = result.to_vector<float>();
    for (size_t i = 0; i < data_vec.size(); ++i) {
        data_vec[i] = data_vec[i] / 255.0f;
    }
    result = Tensor(data_vec, input_shape, 
                    dtype(DataType::kFloat32).device(device));
    
    // 6. 标准化（ImageNet 标准化参数）
    float mean[3] = {0.485f, 0.456f, 0.406f};
    float std[3] = {0.229f, 0.224f, 0.225f};
    
    for (int c = 0; c < input_c; ++c) {
        for (int h = 0; h < input_h; ++h) {
            for (int w = 0; w < input_w; ++w) {
                size_t idx = c * input_h * input_w + h * input_w + w;
                data_vec[idx] = (data_vec[idx] - mean[c]) / std[c];
            }
        }
    }
    
    return Tensor(data_vec, input_shape, 
                  dtype(DataType::kFloat32).device(device));
}
#endif
```

#### 24.2.4 后处理（NMS）

**后处理流程：**
```
1. 解码边界框（从 anchor 格式转换为坐标格式）
2. 应用置信度阈值
3. 非极大值抑制（NMS）
4. 坐标转换（从模型坐标转换为图像坐标）
```

**实现：**
```cpp
std::vector<Detection> postprocess_yolo(
    const std::vector<Tensor>& outputs,
    float conf_threshold = 0.5f,
    float nms_threshold = 0.4f) {
    
    std::vector<Detection> detections;
    
    // 1. 解码所有尺度的输出
    for (const auto& output : outputs) {
        auto decoded = decode_yolo_output(output);
        detections.insert(detections.end(), decoded.begin(), decoded.end());
    }
    
    // 2. 应用置信度阈值
    detections.erase(
        std::remove_if(detections.begin(), detections.end(),
                      [conf_threshold](const Detection& d) {
                          return d.confidence < conf_threshold;
                      }),
        detections.end());
    
    // 3. NMS
    detections = nms(detections, nms_threshold);
    
    // 4. 坐标转换
    for (auto& det : detections) {
        // 从模型坐标转换为图像坐标
        det.bbox = transform_coordinates(det.bbox, image_size);
    }
    
    return detections;
}

### 24.3 性能优化

#### 24.3.1 推理速度优化

**优化策略：**

1. **使用静态图：**
   ```cpp
   // 静态图：图结构固定，执行效率高
   PNNXGraph graph("model.param", "model.bin");
   graph.build();  // 一次构建，多次使用
   ```

2. **禁用梯度计算：**
   ```cpp
   {
       auto guard = no_grad();  // 不构建计算图
       graph.forward();
   }
   ```

3. **批量推理：**
   ```cpp
   // 批量处理多个输入
   Tensor batch_input = stack(inputs);  // (batch_size, ...)
   graph.set_inputs("pnnx_input_0", {batch_input});
   graph.forward();
   ```

4. **算子融合：**
   - 在模型导出时进行算子融合
   - 减少算子调用次数

#### 24.3.2 内存优化

**优化策略：**

1. **内存复用：**
   - 复用中间结果的内存
   - 避免不必要的内存分配

2. **及时释放：**
   ```cpp
   {
       auto outputs = graph.get_outputs("pnnx_output_0");
       // 使用 outputs
   }  // outputs 自动释放
   ```

3. **使用内存池：**
   - 预分配内存池
   - 减少分配开销

#### 24.3.3 CUDA 加速

**CUDA 使用：**
```cpp
#ifdef WITH_CUDA
// 1. 初始化 CUDA
origin::cuda::init();

// 2. 将模型迁移到 GPU
graph.to(Device("cuda:0"));

// 3. 将输入迁移到 GPU
auto input_gpu = input.to(Device("cuda:0"));

// 4. 推理（自动在 GPU 上执行）
graph.set_inputs("pnnx_input_0", {input_gpu});
graph.forward();
#endif
```

**性能提升：**
- GPU 并行计算：显著加速卷积、矩阵乘法等操作
- 内存带宽：GPU 内存带宽远高于 CPU
- 批量处理：GPU 更适合批量处理

**注意事项：**
- 数据传输开销：CPU ↔ GPU 数据传输有开销
- 小批量：小批量时 GPU 优势不明显
- 内存限制：GPU 内存有限
