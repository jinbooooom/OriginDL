## 第20章：PNNX 静态图架构

### 20.1 静态图 vs 动态图

#### 20.1.1 静态图的特点

**静态图架构：**

```
┌─────────────────────────────────────┐
│  静态图特点                          │
├─────────────────────────────────────┤
│                                      │
│  ✅ 图结构固定                       │
│     不能根据输入动态改变              │
│                                      │
│  ✅ 预编译优化                       │
│     可以进行图级别优化                │
│                                      │
│  ✅ 执行效率高                       │
│     不需要动态构建图的开销            │
│                                      │
└─────────────────────────────────────┘
```

**静态图使用流程：**

```
静态图使用流程：
  ┌─────────────────────────────────────┐
  │  1. 加载模型文件                     │
  │     PNNXGraph("model.param", "model.bin")│
  └─────────────────────────────────────┘
              │
              ▼
  ┌─────────────────────────────────────┐
  │  2. 构建计算图                       │
  │     graph.build()                    │
  │     └─> 确定执行顺序                 │
  └─────────────────────────────────────┘
              │
              ▼
  ┌─────────────────────────────────────┐
  │  3. 执行推理                         │
  │     graph.forward()                  │
  │     └─> 按固定顺序执行               │
  └─────────────────────────────────────┘
```

**边连接架构：**

```
              │
              │ 输出 "x"
              ▼
  ┌─────────────────────────────────────┐
  │  节点 B                             │
  │    input_names = ["x", "y"]         │
  │    └─> 包含 "x"，建立连接 A → B     │
  └─────────────────────────────────────┘
              │
              │ 输出 "x"
              ▼
  ┌─────────────────────────────────────┐
  │  节点 C                             │
  │    input_names = ["x"]              │
  │    └─> 包含 "x"，建立连接 A → C     │
  └─────────────────────────────────────┘
```

```
边连接建立流程：
  ┌─────────────────────────────────────┐
```

#### 20.2.4 权重存储

**权重存储方式：**
- 存储在 `.bin` 文件中（ZIP 压缩）
- 每个权重是一个单独的文件
- 文件名：`operator_name.attr_name`

**权重加载：**
```cpp
void PNNXParser::load_weights(const std::string &bin_path,
                              std::vector<std::shared_ptr<PNNXNode>> &nodes) {
    StoreZipReader szr;
    szr.open(bin_path);
    
    for (auto &node : nodes) {
        for (auto &attr_pair : node->attributes) {
            std::string filename = node->name + "." + attr_pair.first;
            
            // 读取权重数据
            std::vector<float> weight_data(size);
            szr.read_file(filename, (char *)weight_data.data());
            
            // 存储到 Attribute 中
            attr_pair.second.data = weight_data;
        }
    }
}
```

**权重格式：**
- 数据类型：float32（f32）
- 存储顺序：按形状展开的一维数组
- 字节序：小端序

### 20.3 PNNXGraph 类设计

#### 20.3.1 图构建流程（init、build）

**构建流程：**
```
构造函数
  ↓
init()  // 解析文件，创建节点
  ↓
build()  // 建立连接，拓扑排序，创建算子
  ↓
Complete  // 图构建完成，可以执行
```

**init() 方法：**
```cpp
bool PNNXGraph::init() {
    // 1. 解析模型文件
    nodes_ = PNNXParser::parse(param_path_, bin_path_);
    
    // 2. 建立节点映射
    for (auto &node : nodes_) {
        node_map_[node->name] = node;
    }
    
    graph_state_ = GraphState::NeedBuild;
    return true;
}
```

**build() 方法：**
```cpp
void PNNXGraph::build() {
    if (graph_state_ == GraphState::Complete) {
        return;  // 已经构建完成
    }
    
    if (graph_state_ == GraphState::NeedInit) {
        init();
    }
    
    // 1. 创建节点连接关系
    create_node_relations();
    
    // 2. 拓扑排序
    topological_sort();
    
    // 3. 为每个节点创建对应的 Operator
    for (auto &node : nodes_) {
        if (node->type != "pnnx.Input" && node->type != "pnnx.Output") {
            node->op = OperatorMapper::create_operator(node);
        }
    }
    
    // 4. 构建输出名称映射
    build_output_name_map();
    
    graph_state_ = GraphState::Complete;
}
```

#### 20.3.3 拓扑排序

**目的：**
- 确定算子的执行顺序
- 确保依赖关系正确
- 支持并行执行优化

**算法：**
```cpp
void PNNXGraph::topological_sort() {
    std::map<std::string, bool> visited;
    std::vector<std::shared_ptr<PNNXNode>> sorted;
    
    // DFS 遍历
    for (auto &node : nodes_) {
        if (!visited[node->name]) {
            topological_sort_dfs(node, visited, sorted);
        }
    }
    
    // 设置执行顺序
    for (size_t i = 0; i < sorted.size(); ++i) {
        sorted[i]->execution_order = i;
    }
}
```

**DFS 辅助函数：**
```cpp
void PNNXGraph::topological_sort_dfs(std::shared_ptr<PNNXNode> node,
                                      std::map<std::string, bool> &visited,
                                      std::vector<std::shared_ptr<PNNXNode>> &sorted) {
    visited[node->name] = true;
    
    // 先处理依赖节点
    for (const auto &input_name : node->input_names) {
        auto input_node = find_node_by_output(input_name);
        if (input_node && !visited[input_node->name]) {
            topological_sort_dfs(input_node, visited, sorted);
        }
    }
    
    // 然后处理当前节点
    sorted.push_back(node);
}
```

#### 20.3.4 执行顺序确定

**执行顺序：**
- 通过 `execution_order` 字段记录
- 拓扑排序后设置
- 执行时按此顺序运行

**执行：**
```cpp
void PNNXGraph::forward() {
    // 按 execution_order 排序
    std::vector<std::shared_ptr<PNNXNode>> sorted_nodes;
    for (auto &node : nodes_) {
        if (node->type != "pnnx.Input" && node->type != "pnnx.Output") {
            sorted_nodes.push_back(node);
        }
    }
    
    std::sort(sorted_nodes.begin(), sorted_nodes.end(),
              [](const auto &a, const auto &b) {
                  return a->execution_order < b->execution_order;
              });
    
    // 按顺序执行
    for (auto &node : sorted_nodes) {
        // 执行算子
        execute_node(node);
    }
}
```

### 21.1 PNNXParser 实现

#### 21.1.1 文件解析（.param 文件）

**解析流程：**
```
打开文件
  ↓
读取 magic number（验证格式）
  ↓
读取算子数量和操作数数量
  ↓
解析每个算子行
  ↓
解析参数、属性、形状
```

**parse_param_file() 实现：**
```cpp
void PNNXParser::parse_param_file(const std::string &param_path,
                                  std::vector<std::shared_ptr<PNNXNode>> &nodes) {
    std::ifstream file(param_path);
    
    // 1. 读取 magic number
    std::string line;
    std::getline(file, line);
    int magic = std::stoi(line);
    if (magic != 7767517) {
        THROW_RUNTIME_ERROR("Invalid PNNX param file");
    }
    
    // 2. 读取算子数量和操作数数量
    std::getline(file, line);
    int operator_count, operand_count;
    std::istringstream(line) >> operator_count >> operand_count;
    
    // 3. 解析每个算子
    for (int i = 0; i < operator_count; ++i) {
        std::getline(file, line);
        auto node = std::make_shared<PNNXNode>();
        parse_operator_line(line, node);
        nodes.push_back(node);
    }
}
```

#### 21.1.2 权重加载（.bin 文件）

**加载流程：**
```
打开 ZIP 文件
  ↓
遍历所有节点
  ↓
遍历节点的所有属性（权重）
  ↓
读取权重文件（operator_name.attr_name）
  ↓
存储到 Attribute 中
```

**load_weights() 实现：**
```cpp
void PNNXParser::load_weights(const std::string &bin_path,
                              std::vector<std::shared_ptr<PNNXNode>> &nodes) {
    StoreZipReader szr;
    szr.open(bin_path);
    
    for (auto &node : nodes) {
        for (auto &attr_pair : node->attributes) {
            const std::string &key = attr_pair.first;
            Attribute &attr = attr_pair.second;
            
            // 计算权重数据大小
            size_t size = 1;
            for (int dim : attr.shape) {
                size *= dim;
            }
            size_t bytesize = size * sizeof(float);
            
            // 构建文件名
            std::string filename = node->name + "." + key;
            
            // 读取权重数据
            std::vector<float> weight_data(size);
            szr.read_file(filename, (char *)weight_data.data());
            
            // 存储到 Attribute 中
            attr.data = weight_data;
        }
    }
}
```

#### 21.1.3 节点创建

**节点创建流程：**
```
解析算子行
  ↓
创建 PNNXNode 对象
  ↓
设置基本属性（type、name、input_names、output_names）
  ↓
解析参数、属性、形状
  ↓
添加到节点列表
```

**parse_operator_line() 实现：**
```cpp
void PNNXParser::parse_operator_line(const std::string &line,
                                     std::shared_ptr<PNNXNode> &node) {
    std::istringstream iss(line);
    
    // 读取基本字段
    std::string type, name;
    int input_count, output_count;
    iss >> type >> name >> input_count >> output_count;
    
    node->type = type;
    node->name = name;
    
    // 读取输入名称
    for (int i = 0; i < input_count; ++i) {
        std::string input_name;
        iss >> input_name;
        node->input_names.push_back(input_name);
    }
    
    // 读取输出名称
    for (int i = 0; i < output_count; ++i) {
        std::string output_name;
        iss >> output_name;
        node->output_names.push_back(output_name);
    }
    
    // 解析参数和属性
    std::string token;
    while (iss >> token) {
        size_t eq_pos = token.find('=');
        if (eq_pos == std::string::npos) continue;
        
        std::string key = token.substr(0, eq_pos);
        std::string value = token.substr(eq_pos + 1);
        
        if (key[0] == '@') {
            // 属性（权重）
            parse_attribute(key.substr(1), value, node);
        } else if (key[0] == '#') {
            // 形状信息
            parse_shape(key, value, node);
        } else {
            // 参数
            parse_parameter(key, value, node);
        }
    }
}
```

#### 21.1.4 属性解析

**属性格式：**
- `@weight=(32,3,6,6)f32`：权重属性
- `@bias=(32,)f32`：偏置属性

**解析实现：**
```cpp
void PNNXParser::parse_attribute(const std::string &key, const std::string &value,
                                 std::shared_ptr<PNNXNode> &node) {
    // 解析形状和类型：如 "(32,3,6,6)f32"
    size_t type_pos = value.find_last_of("f");
    if (type_pos == std::string::npos) return;
    
    std::string shape_str = value.substr(0, type_pos);
    std::string type_str = value.substr(type_pos);
    
    // 解析形状
    std::vector<int> shape = parse_shape_string(shape_str);
    
    // 创建 Attribute
    Attribute attr;
    attr.shape = shape;
    attr.type = (type_str == "f32") ? 1 : 0;
    
    node->attributes[key] = attr;
}
```

#### 21.2.2 依赖关系分析

**依赖关系：**
- 节点 B 依赖节点 A，如果 B 的输入是 A 的输出
- 通过拓扑排序确定执行顺序

**分析：**
```cpp
// 在拓扑排序中分析依赖关系
void topological_sort_dfs(std::shared_ptr<PNNXNode> node, ...) {
    // 先处理依赖节点（输入节点）
    for (const auto &input_name : node->input_names) {
        auto input_node = find_node_by_output(input_name);
        if (input_node && !visited[input_node->name]) {
            topological_sort_dfs(input_node, visited, sorted);
        }
    }
    
    // 然后处理当前节点
    sorted.push_back(node);
}
```

#### 21.2.3 拓扑排序算法

**算法：深度优先搜索（DFS）**

**实现：**
```cpp
void PNNXGraph::topological_sort() {
    std::map<std::string, bool> visited;
    std::vector<std::shared_ptr<PNNXNode>> sorted;
    
    // DFS 遍历所有节点
    for (auto &node : nodes_) {
        if (!visited[node->name]) {
            topological_sort_dfs(node, visited, sorted);
        }
    }
    
    // 设置执行顺序
    for (size_t i = 0; i < sorted.size(); ++i) {
        sorted[i]->execution_order = i;
    }
}
```

**DFS 辅助函数：**
```cpp
void PNNXGraph::topological_sort_dfs(std::shared_ptr<PNNXNode> node,
                                      std::map<std::string, bool> &visited,
                                      std::vector<std::shared_ptr<PNNXNode>> &sorted) {
    visited[node->name] = true;
    
    // 先处理依赖节点（输入节点）
    for (const auto &input_name : node->input_names) {
        auto input_node = find_node_by_output(input_name);
        if (input_node && !visited[input_node->name]) {
            topological_sort_dfs(input_node, visited, sorted);
        }
    }
    
    // 然后处理当前节点
    sorted.push_back(node);
}
```

**结果：**
- 排序后的节点列表：依赖节点在前，被依赖节点在后
- 执行顺序：按排序后的顺序执行

```cpp
std::vector<int> parse_tuple(const std::string &str) {
    // 去除括号
    std::string content = str.substr(1, str.length() - 2);
    
    // 分割
    std::vector<int> result;
    std::istringstream iss(content);
    std::string token;
    while (std::getline(iss, token, ',')) {

}
```

```cpp
std::shared_ptr<Operator> OperatorMapper::create_linear(std::shared_ptr<PNNXNode> node) {
```

```cpp
Tensor attribute_to_tensor(const Attribute &attr) {

    }
    
    // 创建 Tensor

                                     TensorOptions(DataType::kFloat32));
```

```cpp
void PNNXGraph::propagate_outputs(std::shared_ptr<PNNXNode> node) {
    // 找到使用该节点输出的下游节点
    for (const auto &output_name : node->output_names) {
        for (auto &other_node : nodes_) {
            if (other_node == node) continue;
            
            // 检查是否使用该输出
            for (size_t i = 0; i < other_node->input_names.size(); ++i) {
                if (other_node->input_names[i] == output_name) {
                    // 设置输入
                    other_node->inputs[i] = node->outputs[0];  // 简化：假设单输出
                }
            }
        }
    }
}
```

#### 22.1.2 按拓扑序执行算子

**执行流程：**
```
按 execution_order 排序节点
  ↓
遍历排序后的节点
  ↓
收集输入（从上游节点的输出）
  ↓
执行算子
  ↓
存储输出（供下游节点使用）
```

**实现：**
```cpp
void PNNXGraph::forward(bool debug) {
    // 按 execution_order 排序
    std::vector<std::shared_ptr<PNNXNode>> sorted_nodes;
    for (auto &node : nodes_) {
        if (node->type != "pnnx.Input" && node->type != "pnnx.Output") {
            sorted_nodes.push_back(node);
        }
    }
    
    std::sort(sorted_nodes.begin(), sorted_nodes.end(),
              [](const auto &a, const auto &b) {
                  return a->execution_order < b->execution_order;
              });
    
    // 按顺序执行
    for (auto &node : sorted_nodes) {
        // 收集输入
        std::vector<Tensor> inputs;
        for (const auto &input_name : node->input_names) {
            auto input_node = find_node_by_output(input_name);
            if (input_node) {
                inputs.push_back(input_node->outputs[0]);
            }
        }
        
        // 执行算子
        auto outputs = node->op->forward(inputs);
        
        // 存储输出
        node->outputs = outputs;
        
        // 传播到下游节点
        propagate_outputs(node);
    }
}
```
