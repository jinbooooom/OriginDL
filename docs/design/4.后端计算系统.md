## 第10章：OriginMat CPU 后端

### 10.1 Storage 设计

#### 10.1.1 数据拥有者角色

**Storage 的职责：**
- 拥有真正的数据内存
- 管理内存的分配和释放
- 可以被多个 Mat 共享

**设计：**
```cpp
class Storage {
private:
    void* data_;           // 数据指针
    size_t size_;          // 数据大小（字节）
    Device device_;        // 设备类型
    DataType dtype_;       // 数据类型
    
public:
    Storage(size_t size, Device device, DataType dtype);
    ~Storage();
    
    void* data() { return data_; }
    size_t size() const { return size_; }
    Device device() const { return device_; }
    DataType dtype() const { return dtype_; }
};
```

**特点：**
- 不可拷贝（禁用拷贝构造函数）
- 只能移动（支持移动语义）
- 通过 shared_ptr 管理生命周期

#### 10.1.2 内存管理（malloc/free）

**CPU 内存管理：**
```cpp
Storage::Storage(size_t size, Device device, DataType dtype)
    : size_(size), device_(device), dtype_(dtype) {
    
    if (device.type() == Device::Type::kCPU) {
        // CPU 内存分配
        data_ = std::malloc(size);
        if (!data_) {
            THROW_RUNTIME_ERROR("Failed to allocate CPU memory");
        }
    } else if (device.type() == Device::Type::kCUDA) {
        // CUDA 内存分配
        cudaMalloc(&data_, size);
        if (cudaGetLastError() != cudaSuccess) {
            THROW_RUNTIME_ERROR("Failed to allocate CUDA memory");
        }
    }
}

Storage::~Storage() {
    if (device_.type() == Device::Type::kCPU) {
        std::free(data_);
    } else if (device_.type() == Device::Type::kCUDA) {
        cudaFree(data_);
    }
}
```

**内存对齐：**
- 使用 `std::aligned_alloc()` 或 `posix_memalign()` 进行内存对齐
- 提高 SIMD 操作的效率
- 对齐到 32 字节或 64 字节

#### 10.1.3 多 Mat 共享机制

**共享机制：**
```cpp
class OriginMat {
protected:
    std::shared_ptr<Storage> storage_;  // 共享 Storage
    // ...
};
```

**共享场景：**
- 多个 Mat 可以共享同一个 Storage
- 视图操作（reshape、transpose）共享 Storage
- 拷贝操作可以选择共享或创建新 Storage

**示例：**
```cpp
// 创建 Mat
auto mat1 = std::make_unique<OriginMat>(storage, shape, dtype);

// 创建视图（共享 Storage）
auto mat2 = mat1->view(new_shape);  // mat2 共享 storage_

// 深拷贝（创建新 Storage）
auto mat3 = mat1->clone();  // mat3 有独立的 storage_
```

#### 10.1.4 引用计数管理

**引用计数：**
- 使用 `shared_ptr` 自动管理引用计数
- 当最后一个 Mat 释放时，Storage 自动释放
- 无需手动管理内存

**生命周期：**
```
创建 Storage
  ↓
Mat1 持有 shared_ptr<Storage>
  ↓
Mat2 共享 Storage（引用计数 = 2）
  ↓
Mat1 释放（引用计数 = 1）
  ↓
Mat2 释放（引用计数 = 0）
  ↓
Storage 自动释放
```

#### 10.2.3 视图支持（view vs copy）

**视图（view）：**
```cpp
std::unique_ptr<Mat> OriginMat::view(const Shape &shape) const {
    // 检查连续性
    if (!is_contiguous()) {
        THROW_RUNTIME_ERROR("view() requires contiguous tensor");
    }
    
    // 检查元素总数
    if (shape.elements() != shape_.elements()) {
        THROW_RUNTIME_ERROR("view() requires same number of elements");
    }
    
    // 创建视图（共享 Storage）
    auto new_strides = compute_strides(shape);
    return std::make_unique<OriginMat>(storage_, shape, new_strides, dtype_);
}
```

**拷贝（clone）：**
```cpp
std::unique_ptr<Mat> OriginMat::clone() const {
    // 创建新 Storage
    auto new_storage = std::make_shared<Storage>(shape_.elements() * dtype_size_, 
                                                  device(), dtype_);
    
    // 拷贝数据
    copy_data(storage_->data(), new_storage->data(), shape_.elements() * dtype_size_);
    
    // 创建新 Mat
    return std::make_unique<OriginMat>(new_storage, shape_, dtype_);
}
```

**选择策略：**
- **连续张量 + reshape**：使用 view（零拷贝）
- **非连续张量 + reshape**：使用 clone（需要拷贝）
- **需要独立数据**：使用 clone

### 10.3 CPU 算子实现

#### 10.3.1 Element-wise 操作（add、mul）

**加法实现：**
```cpp
std::unique_ptr<Mat> OriginMat::operator+(const Mat &other) const {
    const OriginMat& other_mat = static_cast<const OriginMat&>(other);
    
    // 形状检查
    if (shape_ != other_mat.shape_) {
        THROW_RUNTIME_ERROR("Shape mismatch in addition");
    }
    
    // 创建输出 Storage
    auto result_storage = std::make_shared<Storage>(
        shape_.elements() * dtype_size_, device(), dtype_);
    
    // 执行计算
    auto this_data = data_ptr<float>();
    auto other_data = other_mat.data_ptr<float>();
    auto result_data = static_cast<float*>(result_storage->data());
    
    for (size_t i = 0; i < shape_.elements(); ++i) {
        result_data[i] = this_data[i] + other_data[i];
    }
    
    return std::make_unique<OriginMat>(result_storage, shape_, dtype_);
}
```

**原地加法：**
```cpp
void OriginMat::add_inplace(const Mat &other) {
    const OriginMat& other_mat = static_cast<const OriginMat&>(other);
    
    // 直接修改当前数据
    auto this_data = data_ptr<float>();
    auto other_data = other_mat.data_ptr<float>();
    
    for (size_t i = 0; i < shape_.elements(); ++i) {
        this_data[i] += other_data[i];
    }
}
```

#### 10.3.2 矩阵乘法（matmul、GEMM）

**矩阵乘法实现：**
```cpp
std::unique_ptr<Mat> OriginMat::matmul(const Mat &other) const {
    const OriginMat& other_mat = static_cast<const OriginMat&>(other);
    
    // 形状检查：A (M, K) × B (K, N) = C (M, N)
    if (shape_[1] != other_mat.shape_[0]) {
        THROW_RUNTIME_ERROR("Shape mismatch in matmul");
    }
    
    size_t M = shape_[0];
    size_t K = shape_[1];
    size_t N = other_mat.shape_[1];
    
    // 创建输出
    Shape result_shape{M, N};
    auto result_storage = std::make_shared<Storage>(
        M * N * dtype_size_, device(), dtype_);
    
    // GEMM 计算：C = A × B
    auto A = data_ptr<float>();
    auto B = other_mat.data_ptr<float>();
    auto C = static_cast<float*>(result_storage->data());
    
    for (size_t i = 0; i < M; ++i) {
        for (size_t j = 0; j < N; ++j) {
            float sum = 0.0f;
            for (size_t k = 0; k < K; ++k) {
                sum += A[i * K + k] * B[k * N + j];
            }
            C[i * N + j] = sum;
        }
    }
    
    return std::make_unique<OriginMat>(result_storage, result_shape, dtype_);
}
```

**优化方向：**
- 循环展开
- 缓存分块（tiling）
- SIMD 向量化
- 多线程并行

#### 10.3.3 卷积操作（conv2d、im2col）

**im2col 实现：**
```cpp
std::unique_ptr<Mat> OriginMat::im2col(
    std::pair<int, int> kernel_size,
    std::pair<int, int> stride,
    std::pair<int, int> pad,
    bool to_matrix) const {
    
    // 输入形状：(N, C, H, W)
    // 输出形状：(N*OH*OW, C*KH*KW) 或 (N, OH*OW, C*KH*KW)
    
    // 计算输出尺寸
    int OH = (H + 2 * pad.first - kernel_size.first) / stride.first + 1;
    int OW = (W + 2 * pad.second - kernel_size.second) / stride.second + 1;
    
    // 创建输出
    Shape output_shape;
    if (to_matrix) {
        output_shape = {N * OH * OW, C * kernel_size.first * kernel_size.second};
    } else {
```

```cpp
    std::pair<int, int> pad) const {
```

```cpp
void add_simd(const float* a, const float* b, float* c, size_t n) {
    size_t simd_len = 8;  // AVX 处理 8 个 float
    size_t simd_n = n / simd_len;
    
    for (size_t i = 0; i < simd_n; ++i) {
        __m256 va = _mm256_load_ps(a + i * simd_len);
        __m256 vb = _mm256_load_ps(b + i * simd_len);
        __m256 vc = _mm256_add_ps(va, vb);
        _mm256_store_ps(c + i * simd_len, vc);
    }
    
    // 处理剩余元素
    for (size_t i = simd_n * simd_len; i < n; ++i) {
        c[i] = a[i] + b[i];
    }
}
```

```cpp
template <typename T, typename Op>
__global__ void elementwise_kernel(
    const T *__restrict__ a,
    const T *__restrict__ b,
    T *__restrict__ c,
    size_t n,
    Op op) {

}
```

#### 11.2.4 共享内存使用

**共享内存：**
- 每个 block 内的线程共享的内存
- 访问速度比全局内存快
- 用于数据复用和协作

**示例：矩阵乘法优化：**
```cpp
__global__ void matmul_shared(
    const float* A, const float* B, float* C,
    size_t M, size_t K, size_t N) {
    
    // 共享内存
    __shared__ float tile_A[16][16];
    __shared__ float tile_B[16][16];
    
    // 线程索引
    int row = blockIdx.y * blockDim.y + threadIdx.y;
    int col = blockIdx.x * blockDim.x + threadIdx.x;
    
    float sum = 0.0f;
    
    // 分块计算
    for (int tile = 0; tile < (K + 15) / 16; ++tile) {
        // 加载 tile 到共享内存
        if (row < M && tile * 16 + threadIdx.x < K) {
            tile_A[threadIdx.y][threadIdx.x] = A[row * K + tile * 16 + threadIdx.x];
        }
        if (col < N && tile * 16 + threadIdx.y < K) {
            tile_B[threadIdx.y][threadIdx.x] = B[(tile * 16 + threadIdx.y) * N + col];
        }
        
        __syncthreads();  // 同步
        
        // 计算
        for (int k = 0; k < 16; ++k) {
            sum += tile_A[threadIdx.y][k] * tile_B[k][threadIdx.x];
        }
        
        __syncthreads();  // 同步
    }
    
    // 写入结果
    if (row < M && col < N) {
        C[row * N + col] = sum;
    }
}
```

```cpp
private:
    std::vector<void*> free_blocks_;
    std::map<void*, size_t> block_sizes_;
    size_t pool_size_;
    
public:
    void* allocate(size_t size) {
        // 查找合适的内存块
        for (auto it = free_blocks_.begin(); it != free_blocks_.end(); ++it) {
            if (block_sizes_[*it] >= size) {
                void* ptr = *it;
                free_blocks_.erase(it);
                return ptr;
            }
        }

        cudaMalloc(&ptr, size);
        block_sizes_[ptr] = size;
        return ptr;
    }
    
    void deallocate(void* ptr) {
        free_blocks_.push_back(ptr);
    }
};
```
